# POSDROID #
## Description ##
POS software for android devices
## Features ##
  * Sales
  * Orders
  * Invoices
  * Block Sale
  * Reports
  * Products
  * Workflows
  * Payments
  * UnifiedPOS - UPOS ( Unified Point of Service ) compliant JavaPOS port
  * Application interfaces to Android point-of-service devices.
  * Vendor neutral
